## Status Legend

- **Proposed**: Under discussion
- **Accepted**: Approved and active
- **Deprecated**: Replaced by a newer decision
- **Superseded**: Replaced by [DR-xxx](...)

## Council Voting Legend

Each DR records the council vote:
- S = Security Sentinel
- A = Architecture Guardian
- Q = Quality Auditor
- I = Integration Coordinator

Format: `S:✅ A:✅ Q:✅ I:✅` (all pass) or `S:✅ A:❌ Q:✅ I:⚠️` (mixed)
